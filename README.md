Student Management System (ASP.NET MVC).

A small full-stack web-based Student Management System built using ASP.NET MVC, Entity Framework, and SQL Server. The system is designed to help lecturers and administrators manage students, modules, marks, and academic performance efficiently.


Live Demo(link):
campusflowsystem-h8dqg7amcae8gnbd.southafricanorth-01.azurewebsites.net


Features

 1.Student Management
  Add, edit, delete students
  View student profiles
  Track enrolled modules

 2.Module Management
  Create and manage modules
  Assign students to modules

 3.Marks Management
  Capture student marks
  Edit and update marks
  Automatic performance calculation

 4.Lecturer Dashboard
  View assigned modules
  View enrolled students
  Capture and manage marks

 5.Authentication
  User registration and login system
  Role-based access (Admin / Lecturer / Student)
